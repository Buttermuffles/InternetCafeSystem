=====================================================
  Internet Cafe Monitor - Client PC Installer
=====================================================

HOW TO USE:
  1. Plug this USB into the client PC
  2. Right-click "Install_Client_PC.bat" 
     and select "Run as Administrator"
  3. Follow the prompts
  4. Unplug USB - the agent runs permanently

TO REMOVE LATER:
  1. Right-click "Uninstall_Client_PC.bat"
     and select "Run as Administrator"

FILES:
  ICafeAgent.exe        - The monitoring agent
  Install_Client_PC.bat - One-click installer
  Uninstall_Client_PC.bat - One-click uninstall
  config.json           - Server connection settings

TO CHANGE SERVER IP LATER:
  Edit config.json on this USB before installing,
  or edit C:\Program Files\ICafeMonitor\config.json
  on the client PC after installation.

SERVER: http://192.168.1.200:8000
